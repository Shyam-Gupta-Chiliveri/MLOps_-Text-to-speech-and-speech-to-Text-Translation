# Text-to-Speech (TTS) and Speech-to-Text (STT) with OpenAI

This Jupyter Notebook (`TTS_STT_lesson.ipynb`) provides a comprehensive implementation of OpenAI's Audio APIs. It contains code to programmatically generate lifelike spoken audio from text and to transcribe or translate spoken audio into text using the Whisper model.

## ⚙️ Prerequisites

Before running the notebook, ensure you have the following:

1.  **OpenAI API Key**: You need a valid API key from OpenAI.
2.  **Python Environment**: Python 3.7+ is recommended.
3.  **Required Libraries**:
    * `openai`
    * `python-dotenv`
    * `ipython` (for audio playback within the notebook)
